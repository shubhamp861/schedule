# **App Name**: ScheduleSync

## Core Features:

- Meal Schedule Import: Allow users to import their meal schedule by uploading a JSON file, which serves as the central data source for meal timings and details.
- Automated Meal Reminders: Push personalized browser notifications for breakfast, midday meal, and dinner, precisely at their scheduled times as defined in the imported JSON.
- Nightly Prep Notification: Send a distinct browser notification daily around 11 PM to remind the user about preparing breakfast for the next day, leveraging schedule data.
- Detailed Meal View: Upon clicking any meal notification, navigate to a dedicated page that displays comprehensive information for the selected meal, pulled from the schedule.
- AI Breakfast Prep Assistant: Provide a smart, AI-generated tip or simple idea for preparing the next day's breakfast, drawing context from the meal details in the schedule via a generative AI tool.

## Style Guidelines:

- Primary color: A warm, natural sage green (#A7C270) to evoke freshness and organic nutrition, suitable for a healthy meal planning app.
- Background color: A very light, desaturated green (#F3F5F0) that provides a clean and calming canvas, harmonizing with the primary color while maintaining excellent readability for text.
- Accent color: A lively, golden-yellow (#D9BA26) for call-to-action buttons, notifications, and interactive elements, providing a pleasant contrast and drawing attention.
- Headline font: 'Space Grotesk' (sans-serif) for its modern and organized feel, fitting for scheduling and digital interfaces. Body text font: 'Inter' (sans-serif) for exceptional readability and a clean, objective aesthetic, perfect for meal descriptions and detailed information.
- Utilize simple, clean line icons for meal types (e.g., breakfast, lunch, dinner), notifications, and user settings, ensuring clarity and a consistent, modern visual language.
- Implement a clean, card-based layout for presenting meal schedules, ensuring each meal entry is distinct and easily scannable. Meal details will be displayed in a focused, scrollable view or modal upon notification interaction.
- Incorporate subtle and swift animations for navigation transitions, confirmation feedback on actions like schedule uploads, and when notifications are scheduled or trigger, to enhance user experience without distraction.
